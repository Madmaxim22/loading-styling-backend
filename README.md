# Loading Styling Backend

Backend API сервер для демонстрации загрузки и стилизации данных. Сервер реализует эмуляцию задержки загрузки данных и предоставляет endpoint для получения тестовых новостей.

## Особенности

- Эмуляция задержки загрузки данных (2 секунды)
- CORS поддержка для всех доменов
- REST API endpoint для получения новостей
- Использует Koa.js как основной фреймворк

## Установка

```bash
# Клонируйте репозиторий
git clone <url-репозитория>

# Перейдите в директорию проекта
cd loading-styling-backend

# Установите зависимости
npm install
```

## Запуск

```bash
# Запуск в режиме разработки
npm start

# Сервер запустится на порту 3000
# По умолчанию доступен по адресу: http://localhost:3000
```

## API Endpoints

### GET /data

Возвращает массив тестовых новостей с задержкой в 2 секунды:

```json
[
  {
    "id": 1,
    "title": "Новость 1",
    "body": "Это первая новость из мира кино"
  },
  {
    "id": 2,
    "title": "Новость 2",
    "body": "Это вторая новость из мира кино"
  },
  {
    "id": 3,
    "title": "Новость 3",
    "body": "Это третья новость из мира кино"
  }
]
```

## Зависимости

- [Koa.js](https://koajs.com/) - современный веб-фреймворк для Node.js
- [Koa Router](https://github.com/koajs/router) - маршрутизатор для Koa
- [Koa CORS](https://github.com/koajs/cors) - middleware для обработки CORS
- [Koa Slow](https://github.com/koajs/slow) - middleware для эмуляции медленного ответа
